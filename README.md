* [Passport.js](http://passportjs.org/)
* [Mongoose.js](http://mongoosejs.com/index.html)
* [Pug.js](https://pugjs.org/api/getting-started.html)
* [Socket.io](http://socket.io/)
* [MongoDB](https://www.mongodb.com/)
* [SASS](http://sass-lang.com/)
